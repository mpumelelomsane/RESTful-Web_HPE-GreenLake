# RESTful-APIs using Java Spring boot
In this project I am building a RESTful API using Java Spring Boot.  Java Spring Boot is a tool for building standalone applications that  can run independently of a platform. 
Developing a RESTful web service that can be used to manage a list of employees.

The specific requirements for this service include the following:
It supports the GET, POST, DELETE, and UPDATE commands. 
The GET command returns the full list of employees, while the POST, DELETE, and UPDATE commands affect individual entries.
Each entry has the following fields: first_name, last_name, employee_id, email, and title.
This data is stored on a private cloud server.
